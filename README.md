# Cisco DHCP Snooping


## DHCP Snooping Nedir?

DHCP Snooping, bir ağdaki DHCP trafiğini güvenli bir şekilde yönetmek için kullanılan bir güvenlik özelliğidir. Temel amacı, güvenilmeyen DHCP sunucularından gelen isteklere karşı koruma sağlamaktır.
